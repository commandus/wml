Interface walkthrough

This image points out the major components of the editor interface.



Menu bar contains common pull-down menus used for a wide range of tasks.

Tools panel contains buttons for most common tasks.

Project window displays files.

Elements tree window contains tree of wml deck elements.

Attributes window exposes markup language elements settings to set or edit them.

Code editor window exposes the code for your wml application.

Information window exposes errors list encountered in code.
