Before installing the editor, check that your system meets the following hardware and software requirements and note that for Windows NT and Windows 2000 machines, the user logged in during the installation much have local (machine) administrator rights.

Minimum System Requirements

  • Microsoft Windows 98, ME, NT 4.0, Windows 2000 or higher;
  • Internet Explorer 4.0 or higher;
  • 64 MB RAM;
  • 10 MB available disk space;
  • Display set at 1024x768 resolution, 16-bit color.

Installation procedure

The editor is available in the internet as the downloadable executable file setup.exe. In order to use the editor, you must install the installation file setup.exe. 

Using Windows Explorer, browse to the directory where the installation file setup.exe is located after download.

Double-click the file setup.exe.

The installation program runs, displaying the screen dialogs. 

Read license agreement. If you accepted license agreement, press button "Yes". Then choose the installation directory.

Uninstalling editor

You can uninstall the Toolkit from your computer at any time. To do this, perform the following steps:

  • From the Windows Start menu, select Settings-Control Panel.
  • Choose the Add/Remove Programs icon. 
  • In the scrolling list, select “APOO wml editor” and click Add/Remove. Then confirm that you want to uninstall the editir. The editor files and registry entries are removed from your computer.

Starting the editor

From the Start menu, select Programs - APOO Editor - APOO Editor.

From editor's menu select Options - File associations (shell integration), then select from the list the file types which you would like editor to open when you double-click on then in Explorer.

You can also check program preferences from program's menu, select Options - Preferences.
