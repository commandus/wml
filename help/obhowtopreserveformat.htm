How to preserve text formatting

Text contained within <pre>, <samp>, <code> tags is displayed left-justified.
Note that even though conditions exist where right-left justification is turned off while in smart-justify mode, and text appears left-justified, the underlying justification mode remains smart-justify unless the mode has been explicitly specified otherwise.
Alternately, the text is still tagged as smart-justify, but anything within MONOSPACE tags display as if it were left-justified anyway.
