Generate LIT dialog window

The generate LIT dialog window allows you to compile Open eBook package file (.opf). Select created Open eBook package file in file tree view, then click right mouse button. Popup menu appears:

Click on "Compile ..." menu item. Other way is open .opf file in editor, then press Alt+F9 keys or select File | Generate Lit main menu option.

Generate LIT dialog window appears: 



Click Start button. Take a few seconds while LIT file is compiled. When compilation was finished, you can press Read button to see resulting Microsoft ® Reader eBook. If any fatal errors occurred in source files, and output LIT file was not compiled, the Read button is disabled.

If you want to generate LIT file a little bit quicker, mark check box "automatically start generation next time".
