guide

The <guide> section of the Open eBook package file determines which elements appear on the first page of an eBook:

  • Table of Contents (toc)
  • Content (content)
  • Copyright notices, legal and about this Book (title-page)
  • Rights page (other.ms-copyright)
  • Title page graphic (other.ms-titleimage-standard) 
  • Thumbnail graphic- portrait or landscape (other.ms-thumbimage-standard, other.ms-thumbimage)
  • Cover page graphic- portrait or landscape (other.ms-coverimage-standard, other.ms-coverimage)
  • Chapter tour (other.ms-chaptertour)
  • Link to the first page (other.ms-firstpage)

The <guide> supports only a small group of possible id types and elements derived from the OEB standard.

attributes

id
Provides an element with a unique name within a single document. This is especially useful in works with extensive cross-referencing and linking, such as dictionaries, encyclopedias, or travel guides. Use of ids  simplifies linking and updates.
class
Affiliates an element with one or more classes.
style
Attaches a list of inline (name:value paired) CSS properties to a specific tag, for example: style="name:value;name:value"
Note: Media types other than "text/x-oeb1-css" will be ignored with a warning in a document of type "text/x-oeb1-document".
