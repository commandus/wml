meta
Contains generic meta information relating the Open eBook.
Meta-information is specified with property names and values. This specification does not define any properties, nor does it define how browsers must interpret meta-data.
Meta tells basic host information about the tools that created the source files. In eBooks, meta information about the eBook; this information cannot be accessed through the client software or tools.
Do not confuse this tag with the x-meta tag used in the package file to activate chapter tours and other meta elements.

attributes

name
Specifies the property name. The browser must ignore any meta-data named with this attribute. Servers should not emit WML content containing meta-data named with this attribute.
content
This attribute specifies the property value. This attribute is ignored.
scheme
This attribute specifies a form or structure that may be used to interpret the property value. Scheme values vary depending on the type of meta-data. This attribute is ignored.
