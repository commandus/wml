Installation
Note that for Windows NT/2000/XP machines, the user logged in during the installation much have local machine administrator rights.

1. Unzip all files into a directory.
2. Start setup program.
3. After setup program finishes, start editor.
4. Select Options menu.
5. Choose wml file types page and select the file types which you would like program to open when you double-click on then in Explorer.
