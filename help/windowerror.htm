Information window

The information window show what is wrong in wml deck code. Each line consists of icon represents wml element where error occurred, file name and cursor position and short error description as shown in figure.



To see where errors occurred, click on error line. The code editor window navigates to place where errors exists.

Note that some errors are cascaded. It means you must fix first errors. Please note that if you use re- format feature (Ctrl+F9), all elements with errors destroys.
