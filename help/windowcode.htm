WML code editor window

  • Explorer pop- up menu
  • Explorer info tip

The wml code window allows you to edit the code directly. The code is shown in different colors, these all represent different elements of wml, which makes it easier to read and edit. Changes made to the wml code window are reflected in the element tree window and in the element attributes window after you stops entering code. If you want reflect code changes to other windows, press F9 key.



In the left bottom corner you see current cursor position, in the right bottom corner- currently used code page encoding.
