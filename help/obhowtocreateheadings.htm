How to create headings

Define heading style of varying sizes, with <h1> being the largest and <h6> being the smallest. Mark parts and chapters using tags <h1>, <h2>, <h3>, <h4>, <h5> and <h6>.
Note: In Microsoft Reader, all headings have a default top margin of 1-em. The default bottom margin equals one-third of the mean of the heading em and the body em.
Note: On Windows versions of Reader, headings are not formatted with a default font weight of bold, as they are on web browsers. On Pocket PC versions of Reader, headings do appear bold.
Alternately, the text is still tagged as smart-justify, but anything within MONOSPACE tags display as if it were left-justified anyway.
