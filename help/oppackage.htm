Open eBook package document

To create Open eBook you must create package file and lists all HTML files included in eBook to create with it.

Each eBook publication in accordance to the Open eBook Standard requires a package XML document  (.opf file) to be converted into system- dependent format. 

Package files are XML-style files that like all others XML files are largely lower-case throughout.

To simplify generating LIT files, this software include LIT generator master allows to create LIT file from HTML source file in one step.

You can also download MinutEBook for IE utility from here: http://ebook.commandus.com/ . 

This tool is designed for creation of eBooks .LIT binaries for Microsoft® Reader from web contents of Internet Explorer browser with ease.

After this software is installed, a new tool button will be added to the Tools panel of Internet Explorer browser:

To do .LIT binaries, xHTML file and package document, follow next three steps:

1. Navigate you browser to contents you want to convert to eBook:
  • reports of you company;
  • published phone list;
  • listing of properties;
  • books, references and magazines

2. Press button in the Tools panel and .LIT generation wizard appears. Then simply press [Enter] button 3 or 4 times

3.Read and listen a new created eBook- in some seconds. 

As you know, web contents sometimes are more complex in comparison to PDA display possibilities. So you have to choose different styles of .LIT generation process to skip images, simplify tables or just obtain text contents.
Later you can copy created eBooks to your PDA, or listen to them from a desktop.
