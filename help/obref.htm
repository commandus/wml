Open eBook markup reference

Markup is the basic element of authoring any eBook and consists of subset of HTML elements: tags and their attributes. 

Authoring for Microsoft Reader format is generally similar to authoring for a browser. Most of the tags and attributes will be familiar to anyone who has worked with HTML.

Note: Some tags and attributes does not supported by Windows Reader Version 1.5 or the Pocket PC Reader version.
