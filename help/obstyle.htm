style
Allows an eBook source to contain cascading style information. The style tag encloses rules which, when applied to the tags and attributes contained within the document, determine the final layout for the eBook.
Note: If you add a <style> tag outside the <head> section of an HTML document, it generates a blank page at that location.

Sample:

<style>
 a {text-decoration: none}
 h1 {page-break-before: left; margin-top: 20%; color: green}
 .fl {text-indent: 0em; color: red}
</style>

attributes

type=text/x-oeb1-css|text/css

The type attribute has two values:

  • text/x-oeb1-css for OEB compliant style sheets, and 
  • text/css for non-OEB-compliant, standard CSS style sheets.
