Explorer pop- up menu

You can double click on wml icon in Explorer to open wml file or choose command Edit wml file in pop-up menu if this feature is activated. Figure below demonstrates Explorer pop- up menu.

If you want use this feature, activate it. To do this, start the editor, select File associations in Options menu. Dialog window Shell integration appears. Select WML file types tab. From the list of file types mark up wml files check box, then press Ok button.

 

Explorer info tip

Other suitable feature is info tip for wml files shown below.

Using info tip is practice way to see what size of deck is. If you want use this feature, activate it.

To do this, start the editor. Then select File associations in Options menu. Dialog window Shell integration appears. Select Explorer info tip tab. Press button Register. If you disable this feature, press button Unregister.
