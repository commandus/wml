How to set up ftp proxy

In some reason you can establish an ftp connection to your ISP host thru ftp proxy.

Before you can change an FTP site location, you need to know the FTP server name, port (usually 21), your user name/password and directory path.

To set up ftp proxy settings, follow next steps:

  • On the Ftp menu, click Edit ftp site connection settings (or press Shift-F5)
  • Click Proxy button.

Ftp proxy settings dialog appears.

  • Enter settings

Note: You can try select 5 different proxy types to establish a connection to ftp sites. Set proxy type to none if no proxy settings is required.
