p
The <p> tag establishes both the line wrap and alignment parameters for a paragraph. If the text alignment is not specified, it defaults to left. Empty paragraphs (i.e., an empty element or an element with only insignificant white space) will be considered as significant and does not ignored by browsers.
The browser will insert a line break into the text flow between significant paragraphs.
The <p> tag indents the first line of the paragraph, and creates a one line break at the end of the enclosed text (standard HTML adds two line breaks.)

attributes

align=right|left|center|justify|just
Specifies the text alignment mode for the paragraph. Can be 'left', 'center', 'right', justify or just.
id
Provides an element with a unique name within a single deck.
class
Affiliates an element with one or more classes.
xml:lang
Specifies the natural or formal language in which the document is written.
