Tools panel

The Tools panel contains useful buttons that can be activated by clicking them. 



Each button represents action from left to right:

  • Create new wml deck
  • Open wml file
  • Save current wml deck file
  • Toggle desktop scheme (code editor window only - all windows)
  • Help
  • Move current wml element up
  • Move current wml element down

Note that all listed actions have their shortcut.
