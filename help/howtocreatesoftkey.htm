How to create a soft key

Mobile devices have the ability to have different functions assigned to certain buttons called soft keys. WML allows you to assign certain functions to them.

You must have created a paragraph in the card to create a soft key in. 

Select paragraph (shown as p0, figure below) and press do element button to create new do element.




Click on attribute type and select value from the drop-down menu one of the known types or enter new one. Certain keys are standard on most phones (like prev and accept), while others may not be available on some models.

Assign value to label attribute.

Assign an action for the do element by pressing one of the element buttons:

  • go. This element will link to the specified card when the soft key is pressed;
  • refresh. This element refreshes the variables on the card;
  • prev. This element will link the to the previous card viewed;
  • noop. This element causes nothing to happen when the the soft key is pressed. This can be useful when you wish to override a template that has assigned a function to the soft key.
