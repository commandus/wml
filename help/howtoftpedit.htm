How to edit a file using FTP

Before you can edit files published at FTP site, you need to list one or more ftp sites at the ftp sites list

Then, when you list them:

  • In the Project window expand ftp sites node;
  • Select file type do you want to see in the list in the Filter box;
  • Select ftp site you are just added

  • Browse site, then select and double-click the file to edit

Note: if ftp site settings if not valid, or ftp site does not include any files or directories, you can not to see any files in the ftp storage. Also, if you haven't privileges to store file, you can not (it is possible you are connected ad anonymous user to the host).
