dc:Rights
A copyright statement. This tag is deprecated, use XrML standard instead.

In case of Microsoft ® Reader, add link to copyright information at the cover page using other.ms-copyright reference type in the guide section of the package file.

attributes

id
Provides an element with a unique name within a single document. This is especially useful in works with extensive cross-referencing and linking, such as dictionaries, encyclopedias, or travel guides. Use of ids simplifies linking and updates.
xml:lang
xml language code.
xmlns:dc
Always equal to "http://purl.org/dc/elements/1.0/"

Sample:

<dc:Rights>...</dc:Rights>
