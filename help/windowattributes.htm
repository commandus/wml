WML element attributes window

The wml element attributes window allows you to assign attribute values of the selected element. To do this, select element in wml elements tree window first. Attributes list and their values appears in the wml element attributes window as shown below.



Some elements have combo box or ellipsis button on the right edge of attribute line. Click on the ellipsis button and appropriate attribute editor appears. For instance, select input element and click on ellipsis button of format attribute:
