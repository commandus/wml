How to add or change an FTP location

Before you can change an FTP site location, you need to know the FTP server name, port (usually 21), your user name/password and directory path.

To add a new ftp site location, follow next steps:

  • On the Ftp menu, click Edit ftp site connection settings (or press Shift-F5)

  • Click Add site button.

Ftp connection settings dialog appears.

  • Enter settings

Note: Left user name and password blank both if you want to establish connections as an anonymous user.  Note that root directory is not actually the root user path, '/', but it is initial path where you want to start.

Later you can change an FTP location by clicking Edit on selected location (or by double clicking location in the ftp sites list).
