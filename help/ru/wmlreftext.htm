pcdata
Defines text to be displayed to the user.

comment
WML comments follow the XML commenting style.
Comments are intended to be used by the WML author and are not displayed to the user by the user agent. Note that WML comments cannot be nested.

pre
The pre element tells visual user agents that the enclosed text is "preformatted". When handling preformatted text, user agents may:

  • leave white space intact
  • render text with a fixed-pitch font
  • disable automatic word wrap.
