Help : Basics
How to submit a request
To submit a request, simply click the "New Request" command on the main menu.
How to locate requests
The "My Todo List" command lists requests that you are responsible for. The "Unassigned Requests" command lists requests that have not been assigned. You can use this command to find a request and assign it to yourself. To locate other requests, you can run a tabular report by clicking the "New Tabular Report" command in the "Report Manager".
How to submit, assign, respond to, resolve, close, edit or delete a request
First, you need to locate the request using one of the methods described above. Then, click on the request ID to view the request detail page, where you will see commands for performing all of the operations listed above.
