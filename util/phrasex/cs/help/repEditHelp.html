Help : Report Conditions
The system will search for records that satisfy ALL conditions; i.e. all conditions are combined using AND to search the database.
Date Format
A valid date is an all-digit string delimited by a space, slash, dash or period: " /-." depending on your settings in Control Panel, Regional Settings. The year, month and day must appear in the order specified in the date format string. For example, if the date format is "mm/dd/yyyy", "January 2, 1999" is invalid. It should be "02/01/1999".
How to specify empty values
To specify an empty value for a condition, for example, to list all records that have not been assigned, you can type "ASSIGNED_TO IS NULL" in the "SQL Where Clause" text field. See the following section for further examples and details.
