Help : How to enter HTML fragment text
When entering text for a knowledge base article, a discussion message or a request, you can enter plain text or a HTML fragment. HTML allows you to format the text you enter. It uses tags enclosed in "<" and ">" to represent the formatting information.
For example, "Hello <b>World</b>!" shows up as Hello World! To enter "<" or ">" themselves, you need to type "&lt;" or "&gt;" respectively. To enter the "&" itself, you need to type "&amp;".
