Wapreview installation guide

Follow these steps to install the Internet version of the WAPReview WML Browser:
WARNING:To minimise security risks, the java wricproxy class runs on port 5000 and accepts connections from the local host only. If you are making availabil an internet version of the browser, use the perl version of the proxy server as described here.

  1. Obtain and unzip the latest WAPReview kit from http://wapreview.sourceforge.net (or from http://www.applepiesolutions.com/swap)

  2. cd to wapreview/public_html/src/wbmp2gif, and build the image converter following the instructions in the README.txt file.

  3. Copy the newly build image converter executable to a suitable directory on your server, ensuring that it has appropriate executable permissions.

  4. Copy lwpproxy.cgi and wbmp2gif from wapreview/public_html/src/cgi-bin into a cgi directory belonging to your webserver. Modify wbmp2gif so that it references the wbmp2gif executable you just built. Ensure both the cgi's are executable, and that the perl reference is correct.

  5. Install the perl LWP package available from http://www.cpan.org . Test the package is installed by executing the command 'perl -e "use LWP::UserAgent;"'

  6. copy the following files into the webserver directory from which you want to serve the wapbrowser:
      wapreview/public_html/wapreview.html wapreview/public_html/wapreview.js
      wapreview/public_html/wapreview.jar
      wapreview/public_html/*.jpg
      wapreview/public_html/*.gif

  7. Modify wapreview.html so that the parameters for the gateway and image converter correspond to the URLs for the CGI's you installed in step 4. Also modify the variable 'homebase' to reference the wml page you want to appear when the browser is first launched.

  8. Add the following HTML to the web page from which you want to launch the wap browser, and modify the URL to reference the wapreview.html page you installed in step 6.
    <script>
    function wapreview(){
      open('http://www.YOUR.URL/wapreview.html',
      'aplpibrowser','height=450,width=165,resizable=no,titlebar=no,toolbar=no');
    }
    </script>
    <a href="javascript:wapreview()">start WAP browser</a>

  9. Test your installation.
